## ¿Qué es PyQT5?

En palabras breves PyQT es una biblioteca para crear interfaces graficas para programas o scripts de Python, PyQT es una implementación de QT que es un conjunto de bibliotecas para crear la creación de programas en C++ con interfaz grafica.

## ¿Qué necesitas?

 - Solo Python 3 instalado (preferencia la ultima versión disponible)

**Nota:** Hay incopartibilidad con las biblotecas TLS/SSL en las versiones posteriores de PyQT5.12.4 y posteriores con python 3.7.0 a3.7.3,.

## Instalacion

1. Abre la terminal de windows (puede ser el powershell o la terminal)
2. escribe el codigo ``` pip install pyqt5 ```  

Si necesitas otra version  escribe ``` pip install pyqt5==version ``` ejemplo pip install PyQt5==5.12 (es posible que no te permita instalar la version ya que no se encuentra en producion)

**Nota:** ya salio la version 6 de Qt pero al memento de escribir esto no habia salido para instalarlo solo remplaze 5 por el 6.

3. Despues ejecute ```pip install pyqt5-tools```
4. Cuando termine la instalcióin buscara la siguiente carpeta

C:\Users\rubns\AppData\Local\Programs\Python\Python39\Lib\site-packages\qt5_applications\Qt\bin
